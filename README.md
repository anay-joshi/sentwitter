### Download

```bash
git clone https://github.com/amalshaji/sentwitter
```

### Get twitter API keys from [here](https://developer.twitter.com/en/portal/dashboard)

#### create config.py in `frontend`

```python
# config.py
consumer_key= ''
consumer_secret= ''
access_token= ''
access_token_secret= ''
```

### Spin up the app

```bash
docker-compose up
```

### Open browser and goto `localhost:8501`
